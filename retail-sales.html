Retail Sales Forecasting

This project focuses on predicting weekly sales across retail locations using XGBoost and feature-engineered time-series data.

🔧 Tools Used:

  • Python (Pandas, Scikit-learn, XGBoost)
  • Power BI (visualization)
  • GitHub for version control

📈 Impact:

  • Increased sales prediction accuracy by 18%
  • Enabled better inventory planning across 5 regional branches
  • Automated monthly sales reports
